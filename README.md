# 메인 모듈 (Main Module)

피드, 그리드 형식 피드, 리뷰 추가, 음식점 찾기, 프로필  5가지 화면을 제공하는 모듈

하위 모듈들의 최신 버전을 취합, 유기적으로 잘 동작하는지 확인하는 모듈
